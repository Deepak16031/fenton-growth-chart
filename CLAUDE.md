# Fenton Growth Chart Project Guidelines

## Build Commands
- Setup: `npm install`
- Development: `npm run dev`
- Build: `npm run build`
- Lint: `npm run lint`
- Test: `npm test`
- Single test: `npm test -- -t "test name"`

## Code Style Guidelines
- Use TypeScript for type safety
- Follow ESLint configuration for code quality
- Use functional components with React hooks
- Naming: camelCase for variables/functions, PascalCase for components/interfaces
- Imports: group React, libraries, components, styles; order alphabetically
- Error handling: use try/catch for API calls, provide user-friendly error messages
- Use Tailwind CSS for styling with a calm, minimalist color palette
- Commit messages: use conventional commits format (feat/fix/docs/etc)
